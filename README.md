# Features
* SQL Based Race system with in-game editor to make race tracks.
* Property owning system.
* Ability to own two personal vehicles.
* Freeroam activities; deathmatch, mod shops, ammu-nations hospitals and more!
* Account/Admin/Ban systems.
* Custom content that protects the original gta vc charm.
* Unique maps
* and more!

#

This gamemode has been made with my vision of what vc-mp should have been, activities other than dming just like racing or stunting with lots of other things to explore, pretty balanced and choatic deathmatch with no teams, fair ping/fps limits, toggleable crouch glitching, custom content that protects gta vc's feeling and a very fun and unique map to explore. I always wanted to make a big freeroam gamemode similiar to GTA Online, thinking it would be both fun to play and code it
 and here it is, mostly made using some parts of my old scripts such as ViceRacing and MiamiTDM. An Older version of this gamemode with way less features was hosted by Hanney back in the day while it was fun as it lasted i am not thinking of hosting it as of now. but i would be happy if popular servers use some of the ideas from the script.
